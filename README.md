# clicker_tank_privacy_policy.github.io

Privacy Policy — Clicker Tank

Data collection
Clicker Tank does not collect, store, or share personal data from users.

Offline use
The app operates completely offline; no data is transmitted from the device.

Permissions
The app does not request permissions that access personal data (location, contacts, camera, microphone, storage) or persistent identifiers.

Third parties
No third-party services (for example, analytics, advertising, or sign-in) that may collect data are included.

Changes
If features that collect data are added in the future, this policy will be updated and users will be notified within the app.
